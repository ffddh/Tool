**关于番茄小说广告**
```ruby

```
**关于safari聚合搜索**
```ruby
Qsearch是百度搜索的，Safari默认的百度搜索就可以用
Qsearchddg是DuckDuckgo的，需要更改为DuckDuckgo搜索
推荐使用Qsearch
```
